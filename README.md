# 🛒 Online Shopping Behavior Analysis

A full-stack data analytics project developed using **Flask**, **Python**, **Pandas**, **Plotly**, **Streamlit**, **SQLite**, **HTML**, **CSS**, and **JavaScript**.

This project analyzes customer online shopping behavior and provides interactive visualizations to identify purchasing trends, customer preferences, and key business insights.

# 🚀 Installation

Clone the repository

```
git clone https://github.com/yourusername/OnlineShoppingBehaviorAnalysis.git
```

Move into the project folder

```
cd OnlineShoppingBehaviorAnalysis
```

Install dependencies

```
pip install -r requirements.txt

or

py -m pip install flask flask-login flask-sqlalchemy pandas numpy plotly streamlit matplotlib seaborn werkzeug

```

Run Flask

```
py -m backend.app
```

Open

```
http://127.0.0.1:5000
```

Run Streamlit

```
py -m streamlit run dashboard/streamlit_app.py
```
